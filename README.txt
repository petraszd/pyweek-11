RUNNING THE GAME:
-----------------

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.

Otherwise open a terminal / console and "cd" to the game directory and run:

  python run_game.py


HOW TO PLAY THE GAME:
---------------------

Ball (red one) must be caught with a basket (green one).  In some levels You
can control position of the basket.  In some levels there is forces (red
arrows). You can control power of forces (0% -- 100%).


All actions must be done with the help of mouse.

Basket:

  The basket can be moved when green arrows appears near it. Just press mouse
  and drag it. The basket can only be moved on X axis.


Force:

  The force can be controller by pressing mouse on it and dragging up and
  down.
